# scoop-tokentop

[Scoop](https://scoop.sh) bucket for [tokentop](https://github.com/tokentopapp/tokentop).

## Install

```powershell
scoop bucket add tokentop https://github.com/tokentopapp/scoop-tokentop
scoop install tokentop
```

## Update

```powershell
scoop update tokentop
```

## About

This bucket is automatically updated by the [tokentop release workflow](https://github.com/tokentopapp/tokentop/blob/main/.github/workflows/release.yml) on each new release. SHA256 hashes are computed from the release artifacts and committed here by CI.
